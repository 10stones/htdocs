# 07-后续开发规范

## 分支与发布

```text
Codex -> Feature Branch -> Pull Request -> Review -> Merge into main -> GitHub Actions -> Baidu BOS -> CDN
```

生产始终从 `main` 部署。

## 代码规范

- TypeScript 严格模式。
- 页面放在 `src/app`。
- 可复用 UI 放在 `src/components`。
- 工具函数放在 `src/lib`。
- 不要提交密钥或本地部署配置。
