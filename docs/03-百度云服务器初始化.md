# 03-百度云服务器初始化

## Ubuntu 初始化

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y curl git nginx rsync tar ufw
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs
sudo npm install -g pm2
sudo mkdir -p /var/www/zaneshi/releases /var/www/zaneshi/shared /var/www/letsencrypt
sudo chown -R $USER:$USER /var/www/zaneshi
sudo ufw allow OpenSSH
sudo ufw allow 'Nginx Full'
sudo ufw --force enable
```

## GitHub Secrets

在仓库 Settings → Secrets and variables → Actions 添加：

- SERVER_HOST
- SERVER_USER
- SERVER_PORT
- SERVER_PATH：建议 `/var/www/zaneshi`
- SSH_PRIVATE_KEY
