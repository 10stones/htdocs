# 06-自动部署

## 发布目录

```text
/var/www/zaneshi/current -> releases/<git-sha>
/var/www/zaneshi/releases/<git-sha>
/var/www/zaneshi/shared
```

## GitHub Actions 步骤

1. `npm install`
2. `npm run lint`
3. `npm run build`
4. 打包 `.next/standalone`
5. 通过 SSH 上传到服务器
6. 原子切换 `current` 软链
7. `pm2 reload` 应用
8. `nginx -t && systemctl reload nginx`

## 回滚

服务器保留最近 5 个版本，可执行：

```bash
SERVER_PATH=/var/www/zaneshi bash scripts/rollback.sh
```
